== GhostXPS ==

This is a build of the GhostXPS document conversion engine for
Microsoft Windows. The gxps.exe executable handles files
in the Microsoft XPS format.

These GhostXPS builds are command-line tools. You'll need to invoke
them from an MS-DOS/Windows command prompt. For example:

  gxps.exe tiger.xps
